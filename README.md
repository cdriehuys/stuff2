# Stuff 2

The highly anticipated sequel to the also unfinished "Stuff" project. The goal
is for this to be a tracker for the random things you own.

## Roadmap

- [ ] Basic account management
 - [x] Register
 - [ ] Verify your email
 - [ ] Log in
- [ ] Track items you have
- [ ] Answer useful questions about things you own
  - [ ] When did I buy this?
  - [ ] Where did I get this from?
  - [ ] Is there a warranty for this? How long do I have and how do I make a
        claim?
